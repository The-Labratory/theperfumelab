# Security Policy

## Supported Versions

The Perfume Laboratory / Scentra platform is actively developed and maintained.
Security updates are applied to the latest production version.

| Version | Supported |
|--------|-----------|
| Latest | ✅ |
| Older versions | ❌ |

Users should always run the latest deployed version of the application.

---

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

Please **do not open a public GitHub issue for security problems.**

Instead, report vulnerabilities by contacting the maintainer directly:

Email: security@scentra.world  
or  
Email: lawrence@scentra.world

Please include the following information when reporting:

- Description of the vulnerability
- Steps to reproduce the issue
- Screenshots or proof of concept if available
- Potential impact

We aim to acknowledge security reports within **48 hours** and will work to resolve confirmed vulnerabilities as quickly as possible.

Responsible disclosure is appreciated and helps keep the platform safe for everyone.
